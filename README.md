# Island Echo
A short Zelda-inspired top-down dungeon crawler made in a week.

# Controls
Button | Action
:-: | :-
WASD | Move
Space | Charge/Throw Mace
E | Interact
F11 | Cycle screen size
F2    | Toggle FPS display.
F3    | Toggle quick-debug display.
ESC   | Quit (immediately closes the game).

# Tools Used
- Love2D: Game framework.
- CatPaw: My own game engine.
- Tiled: Level design and map creation.
